Anchor Tag's attributes

attributes pada sebuah HTML digunakan untuk memberikan informasi tambahan tentang tag

the anchor tag takes several important attributes

  • the "href" attribute

    "href" stands for hyperlink references

    The value of the "href" tag is the destination we want our link to go to

    This link has a href value of google.com

  • the "target" attribute

    the target attribute is used to open a link in new tab. By default, an anchor tag will open a link in a same tab

    This link will open in a new tab
