Working With Lists

Introduction

There are two types of list, unordered list and ordered list

Every list item in both ordered or unordered list are wrap in "li" tags

The Ordered List

We can initialize ordered list by wraping all list item with the "ol" tag (ol stands for ordered list)

  1. Item one
  2. Item two
  3. Item three
  4. Item four

The Unordered List

We initialize an unordered list using the "ul" tag (ul stands for unordered list)

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Nesting lists

We can also nesting lists

  • Unordered List 1
  • Unordered List 2
    • Unordered List 2.1
    • Unordered List 2.2
  • Unordered List 3
    1. Ordered List Inside Unordered List 1
    2. Ordered List Inside Unordered List 2
    3. Ordered List Inside Unordered List 3
  • Unordered List 4
